## Regras de uso

### Imagens obrigatórias (fachada, rooftop, localização)
- **OBRIGATÓRIO** usar pelo menos 1 imagem de cada categoria nos criativos
- A IA escolhe qual imagem melhor encaixa em cada cena
- Usar como **image-to-image** (referência) — resultado mais fiel ao real
- Strength recomendada: 0.35-0.45 (manter a maior parte da imagem original)
- Se a cena precisa de um ângulo/composição que não existe no banco, gerar via prompt

### Referências de estilo (CODE-MKT)
- São criativos de OUTRO empreendimento
- A IA analisa: estrutura, cortes, ritmo, posicionamento de texto, tom
- A IA NÃO copia imagens/dados — apenas o formato e estilo
- Usar como guia para: duração de cenas, transições, posição de lettering, tom da narração

### Prioridade
1. Sempre buscar imagem no banco obrigatório primeiro
2. Se não tiver imagem adequada → gerar via IA usando prompt
3. Nunca gerar do zero se existe referência disponível no Drive
